KoreOS

KoreOS is a minimal, clean, security-focused operating system based on Debian 13 Trixie and designed for a polished desktop-first experience.

The live ISO supports both UEFI and BIOS boot, installs through Calamares, and boots into KDE Plasma on Wayland with auto-login for the kore user.

Overview

  • Debian 13 Trixie base
  • Live ISO with Calamares installer
  • KDE Plasma on Wayland

Features

Base System Debian 13 Trixie live ISO with UEFI and BIOS support, installable through Calamares.
Desktop KDE Plasma on Wayland with auto-login for the kore user and a dark Fruitiger Aero-inspired theme.
Appearance JetBrains Mono Nerd Font across the UI, terminal, and bootloader, plus a translucent KoreAero Kvantum setup.
Experience GRUB splash, Plymouth animation, optional boot chime, and a polished live-to-install flow.

Design

The visual language stays darg and crisp, with blue accents, translucent surfaces, and a high-contrast layout that reads cleanly on desktop and laptop screens.

About KoreOS

KoreOS is a Debian-based desktop distribution focused on security, clarity, and a consistent visual identity from boot to login to desktop.

Focus

Security hardening, a clean install path, and a themed desktop stack that keeps the system approachable without losing structure.

Platform Stack

Security UFW, AppArmor, sysctl hardening, fail2ban, auditd, firejail, and lynis.
Storage btrfs by default with zstd compression and noatime, plus ext4, XFS, F2FS, exFAT, NTFS, and EROFS support.
Encryption LVM2 and LUKS/cryptsetup supported in the installer flow.
Hardware VirtualBox, VMware, QEMU guest integration, udev rules, kernel modules, and Vulkan/Mesa driver support.

Live Mode

Boots through a fast GRUB splash, launches Plymouth animation, and can switch into copy-to-RAM or safe mode paths when needed.
