KoreOS

KoreOS is a lightweight, security-focused operating system built by Josh Clark and Steven Quintero.

Josh Clark is a software engineer and cyber security specialist. Steven Quintero is a hardware engineer and cyber security specialist. The project focuses on security, customization, and a clean old-school interface.
